# Customer Segmentation Dashboard (RFM Analysis)

## 📌 Project Overview
This project performs customer segmentation using RFM analysis and visualizes insights through a Power BI dashboard.

---

## 🛠 Tools Used
- Python
- Pandas
- Power BI

---

## 📌 Key Insights
- At Risk customers form the largest customer segment.
- High Score customers generate the highest revenue.
- Loyal customers purchase frequently.
- A small group of customers contributes most revenue.

---

## 🚀 Business Impact
- Helps identify high-value customers
- Supports targeted marketing
- Improves customer retention strategies
